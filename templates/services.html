Services

System components, endpoints, and quick operational checks.

Traffic Monitoring

Captures live packet fields (source IP, destination IP, frame length) and shows packet flow summary in real-time.

Endpoint: /api/traffic/start

Network Inventory

Detects internal vs external IPs observed in traffic and stores device activity history to SQLite for reporting.

Endpoint: /api/inventory/scan

Security Check

Performs a threshold-based anomaly check on packet lengths and saves scan history for quick review.

Endpoint: /api/security/scan

Model Test

Runs a small sample through the trained model pipeline (scaler + model + label encoder) to verify ML setup.

Endpoint: /api/model/test

TCP Reliable, connection-oriented protocol used for secure and ordered data transmission.
UDP Fast, connectionless protocol used for real-time communication like streaming and gaming.
ICMP Network diagnostic protocol used for error reporting and connectivity checks.
HTTP/HTTPS Protocols for web communication; HTTPS adds encryption for secure data transfer.
